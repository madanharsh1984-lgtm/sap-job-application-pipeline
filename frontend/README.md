# Frontend - AI Job Automation SaaS

React-based dashboard (planned).

## Planned Tabs (FSD M4.D)

1. **Dashboard** — Eligible job count, recent jobs
2. **Qualified Jobs** — Filtered job listings
3. **Resume Update** — Upload and manage resumes
4. **Connections** — Manage platform connections (LinkedIn, Naukri, Indeed)
5. **Account Details** — Profile, plan, billing

## Getting Started

```bash
cd frontend
npm install
npm run dev
```
